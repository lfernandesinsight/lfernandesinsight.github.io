Soluções reais construídas para resolver problemas de negócio com dados.

01 — Data Engineering / Visualização

LMD Wait-Time Dashboard — Cidadania Espanhola

Pipeline de dados completo e automatizado analisando tempos de espera de solicitações de cidadania espanhola (Lei de Memória Democrática), com base em dados públicos da comunidade. ETL em Python lê a planilha direto do Google Sheets, trata inconsistências reais (datas mistas, duplicatas, texto livre) e carrega num Postgres gerenciado (Neon) via automação diária no GitHub Actions. Dashboard público no Grafana Cloud com KPIs, detecção de outliers (IQR) e análise de tendência.
